# Quality Criteria: Prospera Squad

## 1. Professionalism & Consistency
- All features must follow the Stitch design system exactly.
- Variable naming in code must be semantic and consistent.

## 2. Visual Excellence
- Dark/Light mode transitions must be smooth with no flash of unstyled content (FOUC).
- Dashboard must be fully responsive (390px to 1440px+).

## 3. Performance & Stability
- Lighthouse scores > 90 for Performance and Accessibility.
- Zero critical bugs in main user flows (Login, Transaction, Dashboard).

## 4. Strategy & Value
- Every feature must have a defined success metric (KPI).
- Tasks must be prioritized by Business Value / Effort ratio.
