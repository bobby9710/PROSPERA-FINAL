# Quality Criteria & Scoring — Prospera Evolution

## 1. UX & Estratégia (Peso 30%)
- [ ] O fluxo principal da página é concluído em menos de 3 passos/cliques?
- [ ] A hierarquia de informações permite entender o propósito da tela em 3 segundos?
- [ ] O conteúdo está focado em valor real para o usuário (evita o óbvio)?
- [ ] Existem caminhos alternativos claros para casos de erro (Edge Cases)?

## 2. UI & Estética Premium (Peso 30%)
- [ ] Os tokens do Stitch (Raios de borda, cores, sombras) foram seguidos 100%?
- [ ] A interface possui profundidade visual (camadas, blurs, glassmorphism)?
- [ ] O Dark Mode e o Light Mode estão equilibrados em elegância e legibilidade?
- [ ] A iconografia é consistente em peso, estilo e tamanho?

## 3. Implementação & Performance (Peso 20%)
- [ ] O código é pixel-perfect em relação ao design final aprovado?
- [ ] O layout é fluido (sem quebrar) entre 320px e 2560px de largura?
- [ ] As animações rodam a 60fps sem engasgos (stuttering)?
- [ ] A semântica HTML e a acessibilidade (Aria-labels) estão corretas?

## 4. Conversão & Polimento (Peso 20%)
- [ ] O CTA principal é o elemento mais visível da tela?
- [ ] O texto utiliza gatilhos mentais adequados ao tom de voz Prospera?
- [ ] Foram removidos todos os rastros de "design amador" ou template genérico?
- [ ] O Veredito final foi emitido com base em evidências visuais e técnicas?
