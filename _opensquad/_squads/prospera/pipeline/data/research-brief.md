# Research Brief — Prospera Evolution

## UX Strategy & Product Design (Paulo Produto)
- **Frameworks:** Double Diamond (Discover, Define, Develop, Deliver), Lean UX, Design Thinking.
- **Key Metrics:** Feature Adoption, User Churn, Conversion Rate (CRO).
- **Best Practices:** Information Architecture must be hierarchical (max 3 levels deep). Mobile-first navigation is essential for accessibility. Clear CTAs (Call to Action) define the success of a page.
- **Vocabulary:** User Flow, Cognitive Load, Job-to-be-Done (JTBD), Information Architecture.

## Premium UI & Art Direction (Aline Aparência)
- **Visuals:** Minimalist design, Glassmorphism (frosted glass, background blur 5-15px), Subtle gradients.
- **Dark Mode:** Avoid #000 (Pure Black), use #121212 for backgrounds to reduce eye strain. Text should be off-white to prevent harsh contrast. Use elevation (lighter surfaces) to show depth instead of shadows.
- **Consistency:** Use fixed design tokens for border-radius, spacing, and typography (Inter font).
- **Aesthetics:** Visual hierarchy through size and color. High-quality imagery and consistent iconography (Material Symbols).

## Front-end Precision (Fabio Front)
- **Frameworks:** React (Vite/Next.js), Tailwind CSS, Framer Motion for animations.
- **Micro-interactions:** Purposeful, subtle, and consistent. Fast response time (<100ms for hover effects).
- **Performance:** Optimize images (WebP), lazy-loading, code-splitting (JS). 60fps animations using GPU-accelerated properties (transform, opacity).
- **Accessibility:** Semantic HTML5, ARIA labels, keyboard focus indicators.

## QA & CRO Critical Audit (Vitor Veredito)
- **Audit Checklist:** Usability Heuristics (Nielsen), WCAG 2.1 Accessibility (Contrast > 4.5:1), Performance (Largest Contentful Paint < 2.5s).
- **CRO Strategy:** A/B Testing, Heatmaps, social proof (testimonials, security badges).
- **Visual Polishing:** Detecting "template-like" patterns, fixing spacing inconsistencies (8pt grid system).
- **Vocabulary:** Visual Inconsistency, Cognitive Friction, Happy Path, Edge Case.
